## ✨ 核心功能

- **所见即所得 (WYSIWYG)**：以优雅的富文本编辑模式直接撰写你的模组，告别繁琐的Markdown语法。
- **TRPG 专属模块**：一键插入怪物属性块、法术卡片、物品卡、提示框和警告框。
- **动态掷骰系统**：通过行内骰子快速计算，或是唤出高级骰子面板进行自定义公式投掷（如 4d6kh3 等）。
- **外部数据导入**：支持从 5e.kiwee 或通过 CSV 导入你自己的本地数据库，极速构建内容。
- **高级排版与样式**：灵活自定义页边距、标题样式、字体大小、背景图片及页面主题。
- **无缝导出**：一键导出为精美的 PDF 格式或内置样式的纯正 HTML 文件，排版完美继承。

## 🚀 快速开始

### 本地部署

```bash
# 1. 安装依赖
npm install

# 2. 启动开发服务器
npm run dev

# 3. 构建生产版本
npm run build
```
